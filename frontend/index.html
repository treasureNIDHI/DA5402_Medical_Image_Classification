📚 User Manual & Documentation

🚀 Getting Started

  1. Select a Model: Choose between "Chest X-ray (Pneumonia Detection)" or "Brain MRI (Tumor Classification)"
  2. Upload an Image: Click the file upload button and select a medical image (JPG or PNG)
  3. Click Predict: Press the "🚀 Predict" button to get instant results
  4. View Results: The prediction confidence and processing time will be displayed

📊 Understanding the Pipeline Tab

  • Pipeline DAG: Shows the 13-stage data processing and training pipeline
  • Green Boxes: Completed stages successfully
  • Metrics: Overall pipeline statistics and model performance
  • Model Accuracy: 94%+ for both classification tasks

📈 Monitoring Dashboard Guide

  • Model Performance: Accuracy, F1-Score, and inference latency metrics
  • System Health: API status, model load status, and resource usage
  • Data Quality: Data validation, drift detection, and class balance
  • Error Tracking: Recent errors and system alerts

🔧 Technical Details

Model Architecture: ResNet-50 (ImageNet pre-trained)

Framework: PyTorch

Optimization: INT8 Quantization (75% size reduction to 25MB)

Deployment: Docker + FastAPI REST API

# Example API Call
curl -X POST "http://localhost:8001/predict" \
  -F "file=@chest_xray.jpg" \
  -F "model_type=pneumonia"

💡 Best Practices

  • ✓ Use high-quality medical images for better accuracy
  • ✓ Ensure images are properly oriented and in focus
  • ✓ Pneumonia model works best with chest X-rays (PA view)
  • ✓ Brain tumor model optimized for T2-weighted MRI images
  • ✓ Model predictions are supplementary; consult qualified medical professionals

❓ FAQ & Troubleshooting

Q: What image formats are supported?

A: JPG, PNG, GIF, and other standard image formats (Max 10MB)

Q: How accurate is the model?

A: 94%+ accuracy for both tasks on test datasets. Always verify with professionals.

Q: What if prediction fails?

A: Check image quality, format, and size. Ensure API is running.
